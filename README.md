# screen_pet

A Flutter project for a screen pet application.

## Getting Started

This project is a starting point for a Flutter application with the following dependencies:

- provider: ^6.0.5
- hive: ^2.2.3
- hive_flutter: ^1.1.0
- uuid: ^4.2.2
- shared_preferences: ^2.2.2
- flutter_local_notifications: ^16.3.2

## Setup

1. Ensure you have Flutter installed on your system
2. Run `flutter pub get` to install dependencies
3. Run `flutter run` to start the application
